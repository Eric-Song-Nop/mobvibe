<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCP task args → SQLite task graph | Agent-controlled task/dependency fields mutate durable graph. |
| CLI-local task body → Gateway projection | Task title/description may be local MCP output but not Gateway projection. |
| Dependency mutation transaction | Partial updates can corrupt `blockedBy`/`blocks`. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-05-01 | Tampering | `TaskBoardService` dependency mutation | mitigate | Use SQLite transaction for create/update/unblock; test bidirectional `blockedBy`/`blocks`. |
| T-02-05-02 | Information disclosure | projection builder | mitigate | Keep title/description/body in `body_local_json`; projection tests assert plaintext absence. |
| T-02-05-03 | Spoofing | task owner/caller | mitigate | Resolve owner from durable team members; caller identity remains router-bound and cannot be overwritten by args. |
| T-02-05-04 | Denial of service | dependency graph validation | mitigate | Reject unknown dependency ids and invalid statuses before mutation to avoid corrupt graph states. |
| T-02-05-05 | Elevation of privilege | task tool policy | mitigate | D-07/D-08 remove leader-only and team-tool confirmation gates for bound team members; retain D-10 structural validation and D-09 unchanged ordinary ACP permission flow. |
</threat_model>
