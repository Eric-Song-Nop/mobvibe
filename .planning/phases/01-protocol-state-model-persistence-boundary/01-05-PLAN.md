## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| WebUI API client → Gateway | Browser code can accidentally send plaintext fields unless API surface is metadata-only. |
| Gateway socket event → WebUI store | Projection from Gateway is stored locally; store must not expand it into content state. |
| Documentation → future implementation | Incorrect docs can cause later phases to violate content boundary decisions. |

| T-05-02-I | Information Disclosure | `team-store.ts` persisted state | mitigate | Persist projection/source refs only; redaction test stringifies stored state. |
